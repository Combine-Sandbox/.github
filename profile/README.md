# Combine Sandbox
**Combine Sandbox** is Half-Life 2 mod that lets you do anything and anywhere.

# Features
- [X] Python
- [X] Physics Gun
- [X] Source game mounting support
- [X] Singleplayer mode support
- [X] RTT Lighting (crispy shadows)
